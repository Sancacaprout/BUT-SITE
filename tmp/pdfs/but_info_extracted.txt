Ce qu’est le terminal
Le terminal est une porte de commande textuelle vers ton système.
Analogie simple :
l’explorateur Windows = tu vois les dossiers
le terminal = tu donnes des ordres au système
Le terminal n’est pas “plus dur”. Il est plus direct. Il t’oblige à comprendre :
où tu es
ce qu’il y a dans le dossier courant
où tu veux aller
ce que tu veux faire au fichier
Les trois idées indispensables
Le dossier courant
C’est l’endroit où tu te trouves maintenant.
Commande : pwd
Le contenu du dossier courant
C’est ce qu’il y a autour de toi.
Commande : ls
Le déplacement
C’est comment tu changes d’endroit.
Commande : cd
Les symboles à connaître tout de suite
Symbole Sens concret
/ la racine du système
\~ ton dossier personnel
. le dossier courant
.. le dossier parent

=== PAGE 6 ===
Chemin absolu et chemin relatif
Chemin absolu : l’adresse complète depuis le début.
Exemple : /home/thomas/but-prepa/01_python_algo
Chemin relatif : le chemin depuis l’endroit où tu es.
Exemple : si tu es déjà dans but-prepa , alors cd 01_python_algo
Analogie :
absolu = adresse postale complète
relatif = “tourne à gauche au bout du couloir”
Commandes de base
Commande Action
pwd afficher le dossier courant
ls lister le contenu
cd dossier entrer dans un dossier
mkdir dossier créer un dossier
cp source destination copier
mv source destination déplacer ou renommer
rm fichier supprimer un fichier
rm -r dossier supprimer un dossier et son contenu
cat fichier afficher le contenu d’un petit fichier
less fichier lire un fichier plus confortablement
history voir l’historique
man commande ouvrir le manuel

Une variable est une boîte étiquetée dans laquelle tu ranges une valeur.
Exemple :
age = 18
prenom = "Thomas"
moyenne = 14.5
Ici :
age contient un entier
prenom contient du texte
moyenne contient un nombre décimal
Les types à connaître tout de suite
Type Python À quoi il sert Exemple
int entier 12
float nombre décimal 12.5
str texte "bonjour"
bool vrai/faux True
print et input
print() sert à afficher.
input() sert à demander une saisie à l’utilisateur.
Exemple :
prenom = input("Ton prénom : ")
print("Bonjour", prenom)
Attention : ce que renvoie input() est du texte.
Donc si tu veux faire un calcul, tu dois souvent convertir :
age = int(input("Ton âge : "))
